# Zombie Survival Simulator

hello everyone

this is my first game ever and my first project in godot

i am complete beginner so this game is very simple but i worked really hard on it and learned many things while making it

i hope you have fun playing it

## about game

this game is simple

you just survive and kill zombies for as long as you can

there is no big story or anything fancy yet

just spawn in shoot zombies and try not to die

i want to keep making this game bigger after this first ship

## controls

* w a s d move
* space jump
* left mouse button shoot
* esc uncapture mouse
* double click mouse capture mouse again
